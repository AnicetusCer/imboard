# Support

Imboard is a one-person project by AnicetusCer, so feedback and bug reports are
welcome, but replies may not be instant.

For support, use whichever route fits best:

- Open a GitHub issue for reproducible bugs or feature requests.
- Email the development alias at `dev.acer.certified955@passmail.net`.
- Use the Ko-fi link if you want to leave a tip or send a short message there.

When reporting a problem, please include your desktop environment, whether you
are using SteamOS or another KDE Wayland desktop, how Imboard was installed, and
the steps that reproduce the issue.
